ABOUT PROGRAMMING, DATA STRUCTURES AND ALGORITHMS USING PYTHON

The contents of this website is based on the online NPTEL course Programming, Data Structures and Algorithms using Python by Prof Madhavan Mukund. This website is created to assist and encourage my friends, classmates and everyone who visits this website to enroll to this course. This website does NOT provide any certificates, so if want to earn your certificate, you have to enroll and complete this course on the NPTEL website. Use this website for assistance as it provides lectures and video lectures all in one place.
